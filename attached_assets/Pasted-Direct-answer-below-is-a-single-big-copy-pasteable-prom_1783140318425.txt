Title: Add "Server Hosting" mode — create, run, and manage headless Minecraft servers (prototype)

Background
- Repo: MojoLauncher (Android client launcher). Current launcher downloads Minecraft clients and launches a client JVM on Android devices.
- Goal: Add a server-hosting mode so the app (or a prototype service) can create and run headless Minecraft server instances (vanilla/Paper), manage their lifecycle, show console logs, and perform backups. This is a prototype not a production-ready multi-tenant hosting panel. The focus is: per-instance directory, download server jar, start/stop/nogui lifecycle, console streaming, save/backup, and simple UI/commands.

High-level success criteria (MVP)
1. Create a server instance with a name and chosen server version (vanilla or Paper).
2. Download the server jar into the instance folder and verify checksum (basic verification).
3. Start the server process using a JVM invocation that runs the jar in nogui mode, piping stdout/stderr into the app logs.
4. Provide a UI screen that shows the server console log in real time and accepts admin commands (send "stop" to shut down).
5. Implement save-all before shutdown, and create a zipped backup of the world folder on stop.
6. Store instance metadata (port, memory settings, path) in the existing instances system.
7. Document how to run and test the prototype locally.

Minimal Java example: ServerProcessManager (drop-in prototype)
- Basic start/stop and console streaming skeleton:

```java
// ServerProcessManager.java (prototype)
public class ServerProcessManager {
  private Process process;
  private Thread stdoutThread;

  public void start(File instanceDir, String jarName, String xms, String xmx) throws IOException {
    List<String> cmd = Arrays.asList(
      "java", "-Xms" + xms, "-Xmx" + xmx, "-jar", jarName, "nogui"
    );
    ProcessBuilder pb = new ProcessBuilder(cmd);
    pb.directory(instanceDir);
    pb.redirectErrorStream(true);
    process = pb.start();

    InputStream in = process.getInputStream();
    stdoutThread = new Thread(() -> {
      try (BufferedReader r = new BufferedReader(new InputStreamReader(in, StandardCharsets.UTF_8))) {
        String line;
        while ((line = r.readLine()) != null) {
          // forward line to log storage / UI
          System.out.println("[SERVER] " + line);
        }
      } catch (IOException e) {
        e.printStackTrace();
      }
    });
    stdoutThread.start();
  }

  public void sendCommand(String cmd) throws IOException {
    if (process == null) return;
    OutputStream os = process.getOutputStream();
    os.write((cmd + "\n").getBytes(StandardCharsets.UTF_8));
    os.flush();
  }

  public void stopGraceful(long timeoutMs) throws IOException, InterruptedException {
    if (process == null) return;
    sendCommand("save-all");
    Thread.sleep(1000);
    sendCommand("stop");
    if (!process.waitFor(timeoutMs, TimeUnit.MILLISECONDS)) {
      process.destroy();
    }
  }
}
```
